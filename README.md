# Google口コミ収集アプリ - i.so-ji（アイソウジ）

## 概要

お客様のアンケート回答から口コミ文を自動生成し、Googleマップへの投稿をサポートするWebアプリです。お客様のスマホとスタッフのスマホが連携して動作します。

---

## 全体フロー

### お客様のスマホ
1. **星評価**（1〜5星）
2. **サービス選択**（複数選択可 / 12種類）
3. **ご利用詳細**（初回 or リピート / お悩み・きっかけ選択 + その他自由記載）
4. **アンケート**（はい/いいえ形式の質問 + その他自由記載）
5. **送信** → 6桁のセッションコードが表示される
6. **待機** → スタッフが口コミを編集・確定するまで自動で待機
7. **口コミ確認** → スタッフが確定した口コミが自動で届く
8. **Googleマップ投稿** → ボタンを押すと口コミがクリップボードにコピーされ、Googleマップの投稿画面が直接開く
9. **完了** → 星の数の案内 + 貼り付け手順 + お礼メッセージ

### スタッフのスマホ
1. **セッションコード入力**（お客様から伝えられた6桁）
2. **お客様の回答確認** → 星評価・サービス・きっかけ・アンケート回答が一覧表示
3. **口コミ編集** → アンケート内容から自動生成された口コミ文を確認・編集
4. **プレビュー** → お客様に画面を見せて内容確認
5. **確定** → お客様のスマホに口コミ文を送信

## 口コミ自動生成ロジック

1. **冒頭文** - 星評価（4星/5星/それ以外）× 初回/リピートで出し分け。選択した全サービス名を含む
2. **お悩み自由記載** - 「その他」に入力された文章をそのまま使用
3. **サービス固有文** - 各サービスのお悩みタグでフィルタリングした口コミテンプレートから選択（複数サービス対応）
4. **アンケート回答文** - 「はい」と答えた質問ごとに5パターンからランダム選択、順序もシャッフル
5. **アンケート自由記載** - 「その他」に入力された文章をそのまま使用
6. **締め文** - 星4以上の場合、初回/リピートで出し分け

## 技術構成

### ファイル構成
```
index.html      ... アプリ本体（HTML/CSS/JS 全て1ファイル）
gas_setup.js    ... Google Apps Script（スプレッドシート記録用）
README.md       ... このファイル
```

### 通信方式
- **Firebase Realtime Database** - お客様のスマホとスタッフのスマホ間のリアルタイム通信
  - REST API（PUT/GET）でデータ読み書き
  - Server-Sent Events（SSE）でリアルタイム監視
- **Firebase未設定時** - localStorage + storageイベント（同一端末の別タブでのみ動作）

### 外部連携
- **Google スプレッドシート** - Google Apps Script経由でアンケート結果を自動記録
- **Google マップ** - writereview URL形式で口コミ投稿画面を直接表示

## 設定項目（歯車ボタンから）

| 項目 | 説明 |
|------|------|
| 店舗名 | ヘッダーや口コミ文に使用される店舗名 |
| Google マップの口コミURL | 口コミ投稿画面を直接開くURL |
| 業種 | 店舗の業種（表示用） |
| Firebase Realtime Database URL | 異なる端末間の通信に使用 |
| スプレッドシート記録URL | Google Apps ScriptのウェブアプリURL |
| アンケート質問 | はい/いいえ形式の質問（追加・削除・編集可能） |

## セットアップ手順

### 1. Google マップ口コミURL
- Googleで店舗名を検索 → 「クチコミを書く」を右クリック → リンクのアドレスをコピー
- または Google ビジネスプロフィール → 「クチコミを依頼」からリンクを取得

### 2. Firebase Realtime Database
1. [Firebase Console](https://console.firebase.google.com/) でプロジェクト作成
2. Realtime Database を作成（ロケーション：シンガポール / テストモード）
3. データベースURLを設定画面に登録

### 3. Google スプレッドシート記録
1. 新しいスプレッドシートを作成
2. 1行目にヘッダーを入力：`日時 / 星評価 / サービス / 初回・リピート / きっかけ / きっかけ（その他）/ アンケート（はい）/ アンケート（いいえ）/ 自由記載`
3. 「拡張機能」→「Apps Script」→ `gas_setup.js` のコードを貼り付けて保存
4. 「デプロイ」→「新しいデプロイ」→ 種類「ウェブアプリ」→ アクセス「全員」→ デプロイ
5. 表示されたURLを設定画面に登録

### スプレッドシートに記録される内容

| 日時 | 星評価 | サービス | 初回・リピート | きっかけ | きっかけ（その他） | アンケート（はい） | アンケート（いいえ） | 自由記載 |
|------|--------|---------|--------------|---------|-----------------|------------------|-------------------|---------|
| 2026/04/01 9:00 | 5 | エアコン、浴室 | 初めて利用 | カビ・臭い、水垢 | | スタッフ対応、仕上がり | 料金 | |

## 注意事項

- **Googleマップへの自動投稿は不可** - Googleのセキュリティ仕様上、外部から自動投稿はできません。クリップボードにコピー → お客様が貼り付けて投稿する流れです
- **Firebase テストモードは30日で期限切れ** - 期限前にセキュリティルールを更新する必要があります
- **口コミの自由記載部分はそのまま使用** - お客様が入力した「その他」の文章はテコ入れせずそのまま口コミに反映されます
